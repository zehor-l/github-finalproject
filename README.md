## Paradise Nursery Shopping Cart Application

Paradise Nursery is an online plant shop web application that allows users to browse houseplants, add them to a shopping cart, and manage cart items dynamically. Features include:

- Browse plants by categories
- View plant details: image, name, price
- Add plants to shopping cart
- Increment/decrement quantities and calculate totals
- Interactive navigation bar with Home, Plants, and Cart links
